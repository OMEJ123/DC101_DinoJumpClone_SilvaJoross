## üéÆ Game Description

In this game, you control a dinosaur that automatically runs forward.  
Your goal is to **jump over incoming obstacles** and score as high as possible.  
The game ends when the dinosaur collides with an obstacle.

Features:
- Jump sound effects
- Random obstacle sizes
- Score & High Score (saved using LocalStorage)
- Restart screen after game over
- Animated background and pixel-style design
